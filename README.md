# student-salary-prediction
A Machine Learning project that predicts student salary based on age, gender, education level, job title, and years of experience using Linear Regression.

# Student Salary Prediction using Machine Learning

# Project Overview
This project predicts student salary using Machine Learning. It is built with Python and Scikit-learn using the Linear Regression algorithm.

# Features
- Data Loading and Exploration
- Missing Value Handling
- Label Encoding
- One-Hot Encoding
- Feature Scaling
- Linear Regression Model
- Model Evaluation
- Actual vs Predicted Salary Visualization
- User Salary Prediction

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

# Dataset Features
- Age
- Gender
- Education Level
- Job Title
- Years of Experience
- Salary (Target)

# Model Evaluation
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
